# React Shopping Cart

A multi-page e-commerce shopping cart application built with React and React Router. Products are fetched live from the [FakeStore API](https://fakestoreapi.com). 

---

## Features

- Home, Shop, and Cart pages with client-side routing via React Router
- Products fetched from the FakeStore API
- Product cards with title, image, price, and quantity controls
- Type in a quantity directly or use the + / - buttons on each product
- Navbar cart badge updates in real time as items are added or removed
- Adjust quantities or remove items from the Cart page
- Core logic and UI interactions tested with React Testing Library

---

## Getting Started

### Prerequisites

- Node.js v18+
- npm or yarn

### Installation

```bash
git clone https://github.com/your-username/react-shopping-cart.git
cd react-shopping-cart
npm install
npm run dev
```

The app will be available at `http://localhost:5173`.

---

## Running Tests

```bash
npm test
```
